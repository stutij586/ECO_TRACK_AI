# 🌍 EcoTrack AI

EcoTrack AI is a sustainability-focused web application that helps users estimate their carbon footprint and receive eco-friendly recommendations.

## 🚀 Features

- Carbon Footprint Calculator
- Virtual Eco Tree Growth
- Sustainability Impact Summary
- Carbon Reduction Simulator
- Eco Battle Dashboard
- AI Eco Coach Recommendations
- Achievement Badge System

## 🛠️ Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Railway Deployment

## 🌱 How It Works

Users enter:
- Vehicle Type
- Daily Travel Distance
- Electricity Consumption

The application calculates a carbon score and provides:
- Eco Level
- Virtual Tree Growth
- Sustainability Summary
- Personalized Eco Tips
